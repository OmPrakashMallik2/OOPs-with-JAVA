
Java Source File Structure-Introduction :
    - Only one public class is allowed in a java file.
    - Name of the java file should be as the name of public class
    - if the public class is not available then named the file on any class name.
    - it is a incorrect concept that class which contains main method should be file name.

Java Source File Structure - import statment :
    - when ever we are importing a package all classes and interfaces by default available
      but not subpackage classes if you want to use subpackage class compulsary import statment
      must be util subpackage level.
